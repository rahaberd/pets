News Aggregator

Live financial news pulled from RSS feeds and NewsAPI with WebSocket streaming.

About this tool

The News Aggregator is a standalone FastAPI application with its own frontend. It runs as a separate server and is not embedded here.

Dual-source ingestion

Fetches articles from configurable RSS feeds (BBC, Reuters) and optionally NewsAPI.

Real-time WebSocket streaming

New articles are pushed to all connected clients as they are ingested.

Persistent SQLite storage

Articles are deduplicated by URL and stored locally via SQLAlchemy.

Background polling

Automatic re-fetch every 15 minutes (configurable via .env).

To launch the News Aggregator:

cd news-aggregator pip install -r requirements.txt cp .env.example .env uvicorn app.main:app --reload
Open News Aggregator

Tech Stack

  • FastAPI Web framework + REST API
  • Uvicorn ASGI server
  • SQLAlchemy ORM / SQLite persistence
  • httpx Async HTTP client
  • feedparser RSS parsing
  • WebSockets Real-time push delivery
